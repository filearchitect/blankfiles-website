# @filearchitect/blankfiles-mcp

MCP server for [blankfiles.com](https://blankfiles.com), backed by the Blank Files API.

## Install

```bash
npm install -g @filearchitect/blankfiles-mcp
```

## Run

```bash
blankfiles-mcp
```

Optional:

- `BLANKFILES_BASE_URL` (default `https://blankfiles.com`)

## Tools

- `list_blank_files`
- `files_by_type`
- `file_by_category_type`
